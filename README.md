#  끼리끼리

## :loudspeaker: 사이트 소개
프로젝트와 스터디를 온라인에서 만나서 진행하는 사이트입니다.

## :checkered_flag: 사이트 목적
새롭게 개발자를 시작하려는 경험이 없는 사람들이 경험도 쌓고  
동시에 공부도 할 수 있는 사이트를 만드는 것이 저희의 목표였습니다.
